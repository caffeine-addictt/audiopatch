# Quick Start

_This is an example of how you can set audiopatch locally.
To get a local copy up and running, follow these simple steps._

## Prerequisites

- GNU/make
- Rust
- Docker

## Installation

1. Clone the repo
2. Run `make dev`
3. Visit at `http://localhost:3000`
